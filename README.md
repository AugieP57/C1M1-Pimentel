# Embedded Statistics Analyzer

Author: Augustine Pimentel

This program performs statistical analysis on an array of unsigned char values.

## Functions Implemented

- find_mean()
- find_median()
- find_maximum()
- find_minimum()
- sort_array()
- print_array()
- print_statistics()

## Compilation

Compile using GCC:

gcc stats.c -o stats

## Run

./stats

## Output

The program will display:

- original array
- sorted array (largest to smallest)
- mean
- median
- maximum
- minimum
